工程名称：入门工具
这是本空间的入门工具，目前本空间所有项目都在此环境下进行编程。操作系统是ubuntu.

 dev-tools
介绍了本空间需要用到的基础软件设施以及部分安装步骤。
1. gradle
  类似maven,用来构建Java工程。
  详细看看http://www.gradle.org/

2. ImageMagick
  图像处理软件。
  处理图片，用户网站的图片处理

3.JDK
  本空间主要是Java代码，需要安装
  jdk-7u21-linux-x64.tar.gz

4. MongoDB
  用来存储非钱相关的数据库。

5. PostgreSQL
  关系数据库

6. scripts
  python脚本环境安装.用来支撑本空间需要用到的自动部署项目

7. search
  Lucene索引查看器。
  
8.thrift
  服务器后台通信模块

9.tomcat
  web前端
